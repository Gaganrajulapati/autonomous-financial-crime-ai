# Autonomous Financial Crime AI

AI-powered financial crime detection platform built using Python, Machine Learning, and Graph Analytics.

## Features

• Fraud detection using Isolation Forest  
• Fraud network analysis with NetworkX  
• Money flow visualization (Sankey diagram)  
• Fraud alert monitoring  
• Suspicious account detection  
• Fraud investigation dashboard (Streamlit)

## Tech Stack

Python  
Pandas  
Scikit-learn  
NetworkX  
Plotly  
Streamlit  

## Dashboard

The system provides an interactive investigation dashboard for financial crime monitoring.

## Use Cases

Banking fraud detection  
Money laundering investigation  
Transaction monitoring
